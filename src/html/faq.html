У всех select (кроме первого) сейчас стоит disabled. Они становятся доступными по очереди после заполнения предыдущего (см. комментарии в figma). Стили, соответственно, прописаны для обычного состояния и для disabled.

1. Дилерский центр

Выбор дилерского центра

Сделано с помощью select2. Поле сразу активно.
В option важно прописать все data (data-name, data-address, data-img). Они используются в js-коде для красивого вывода select. В data-img указывается ссылка на картинку с иконками марок.

Карта

Карта открывается по клику на "Показать карту" (js-open-map).
У кнопки 3-и состояния:
data-status="not" - карта еще не запущена (карта инициализируется при первом открытии блока),
data-status="open" - карта открыта,
data-status="close" - карта закрыта.

Карта реализована чз Яндекс Карты API, там прописаны все точки.

Задумка в том, что пользователь должен иметь возможность выбрать точку (адрес центра) на карте (подробнее см. в фигме).

В файлах загружена иконка для выбранного дилерского центра - images/icons/Map_marker-active.svg. Нужно будет ее прописать у выбранного элемента, вместо стандартного.

***

После выбора ДЦ, надпись должна поменяться с "Выберите адрес" на "Вы выбрали адрес"

2. Выбор мастера и даты

Кнопка открытия выбора

Изначально поле недоступно для вывода. Становится доступно после выбора дилерского центра. По сути это просто кнопка (btn-choise-master). При клике на кнопку (после того как она становится активна), открывается информация для заполнения, а сама кнопка скрывается.

Изначально у кнопки добавлен класс btn-choise-master--disabled, чтобы показать, что кнопка не активна. После того как выбран адрес, кнопка должна стать активной и класс необходимо убрать.

После того как будет выбрана вся информация (дата, место, время), кнопку необходимо снова показать (при этом блок с выбором убрать) и добавить класс btn-choise-master--active + передать туда информацию о выбранном мастере, дате и времени (ниже примеры страниц по шагам).

Слайдер с датами

Слайдер с датами - внутри слайдера радио-кнопки для выбора даты.
По умолчанию выбран первый доступный день (checked), предыдущие все disabled. Так же по умолчанию слайдер начинается с первой доступной даты (минус 1) - вычисляется в js.

Блок с выбором мастера и времени

Должен подгружаться в зависимости от выбранного дня.
Внутри сделаны радио-кнопки для выбора нужного времени.

***

После выбора даты, мастера и времени, надпись в заголовке должна поменять с "Выберите мастера и время" на "Вы выбрали мастера и время".

Кнопка "Далее"

У кнопки "Далее" сейчас стоит disabled. Когда кнопка становится активной (после заполнения всех полей), у кнопки убирается disabled и она становится красненькой (в стилях прописано).

Input-ы

Сначала доступно поле "Имя" для заполнения, а у поля "Телефон" и у кнопки "Далее" стоит disabled.

После того, как поле "Имя" заполнено (и прошло валидацию), к нему должен добавляться класс contact-info__input--validity (стандартное тоже подойдет - прописано свойство .contact-info__input:valid), а поле "Телефон" должно стать доступно (необходимо убрать disabled).

У input-а "Телефон" изначально (когда оно НЕ активно) выводится placeholder "Номер телефона", а когда оно становится доступно - "+ 7 (___) ___ __ __", поэтому прописан дополнительный label (он автоматически убирается (display:none), когда поле перестает быть disabled.

После того как поле "Телефон" заполнено (и прошло валидацию), к нему должен добавляется класс contact-info__input--validity, а у кнопки "Далее" необходимо убрать disabled.

***

У input-ов дополнительно указан label - он нужен для отображения статуса (тк у input нет псевдо-элементов before/after).

PopUp-ы

Список PopUp-ов:

  • PopUp с кнопки "Поддержка" - форма обратной связи - можно вызвать с любой страницы, кликнув на кнопку "Поддержка" в подвале.
  • PopUp успешной отправки сообщения на странице об успехе - success_popup.html (клик по кнопке "Завершить" (для примера).
  • PopUp успешной отправки сообщения на странице об ошибке - error_popup.html (клик по кнопке "Отправить" (для примера).

PopUp-ы вызываются с помощью конструкции типа:
<a href="#id-popup" class="js-call-popup">link</a>

Так же можно вызвать PopUp принудительно (например, по какому-то другому событию) с помощью функции в js:
PopUpShow(null, '<a href="#id-popup"></a>');
где ссылка содержит id popUp-а.
